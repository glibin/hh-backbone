Классический MVC

Model / Модель

Набор данных с бизнес-логикой

View / Представление

Отображение модели (ui, forms)

Controller / Контроллер

Контролирует пользовательский ввод, изменяет модель и вид

Что в вебе?

Различные имплементации

  • - MVVM (Model-View-ViewModel)
  • - MVP (Model-View-Presenter)

По факту

  • - Модель отвечает за данные с бизнес-логикой
  • - Представление отвечает за отображение модели и взаимодействие с пользователем (беря на себя функции контроллера)
  • - плюс дополнительная обвязка для связи модель-представление

Варианты

  • - Knockout.js - MVVP / 13Kb
  • - SproutCore
  • - Dojo toolkit
  • - ExtJS (Sencha JS)
  • - Backbone - 5.6Kb
  • ...
